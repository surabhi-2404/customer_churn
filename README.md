## Project Overview

This project analyzes customer churn data and builds multiple machine learning models to predict whether a customer will leave
a telecom service. The notebook demonstrates data preprocessing, model training, evaluation, and comparison of different algorithms.
The goal is to understand how different models perform on the same dataset and identify which provides the best predictive accuracy.

## Features

Data preprocessing and cleaning<br>
Feature selection and transformation<br>
Train–test splitting with different ratios<br>

## Implementation of multiple ML algorithms:<br>
Linear Regression (for numeric prediction)<br>
Logistic Regression (Simple & Multiple)<br>
Decision Tree Classifier<br>
Random Forest Classifier<br>

## Model evaluation using:<br>
Accuracy Score<br>
Confusion Matrix<br>
RMSE (for regression)<br>

## Models Implemented
Model	Type	Purpose<br>
Linear Regression	Regression	Predict monthly charges<br>
Logistic Regression	Classification	Predict churn (single feature)<br>
Multiple Logistic Regression	Classification	Predict churn (multi-feature)<br>
Decision Tree	Classification	Rule-based churn prediction<br>
Random Forest	Classification	Ensemble learning prediction<br>

## Dataset Features Used

tenure → Duration of customer subscription<br>
